# azurepy_AINABI
Proyecto desarrollado originalmente en el primer semestre de 2025 como parte de la materia Proyecto Profesional I de la Universidad Nacional de General Sarmiento. Migrado a este repositorio público para demostración de competencias.
Mi principal contribución fue el desarrollo del modulo de Regresion Lineal donde se crea un DataSet en formato CSV con distintas columnas para representar un archivo hisórico de empleados. Este archivo se crea en generar_synthetic_training_data.py de forma aleatoria siguiendo determinadas reglas dadas por el usuario, el cual sería un analista de datos, para evitar generar sesgos innecesarios intentando abarcar todas las situaciones posibles y hacer el DataSet más apegado a la realidad. Luego en regresion.py, usando Scikit Learn, NumPy y Pandas se extraen los datos necesarios para entrenar un modelo de Machine Learning (Regresión Líneal) utilizando el CSV generado anteriormente. Al terminar el entrenamiento se guarda un archivo pkl para poder reutilizar el modelo y no necesitar re-entrenar el modelo cada vez que se requiera realizar una predicción. También me encargue de implementar y testear los endpoints de app.py, además de dar apoyo en tareas del modulo de K-Means.
